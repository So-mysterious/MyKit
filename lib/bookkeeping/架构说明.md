## 目录结构
- **actions/**: [服务端] 用于 Server Components 的 Actions，处理数据库写操作（CRUD）。
- **cache/**: [客户端] 客户端全局缓存与状态管理（Context + SWR-like logic）。
- **parsers/**: [工具] 各种格式的流水导入解析器。
- **utils/**: [工具] 通用工具（含 formatters.ts 导出格式化）。
- **logic.ts**: [核心] 复式记账核心算法（如余额计算、试算平衡验证）。
- **useColors.ts**: [Hooks] 统一的颜色管理 Hook。
- **useSettings.ts**: [Hooks] 全局设置 Hook。
